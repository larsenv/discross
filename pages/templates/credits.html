Services

Discross utilizes the following services for its features:

  • AccuWeather - Weather information
  • Associated Press - News articles
  • Domino's - Pizza ordering integration
  • ESPN - Sports scores and updates
  • Rotten Tomatoes - Movie information
  • Stooq - Stock market data
  • Yahoo Finance - Currency exchange rates
  • Zap2it - TV guide listings

Dependencies

The following software libraries and dependencies are used in Discross:

  • @sentry/node - Error monitoring
  • async-lock - Resource locking
  • bcrypt - Password hashing
  • better-sqlite3 - Database management
  • discord-emoji-converter - Emoji handling
  • discord.js - Discord API interaction
  • dotenv - Environment configuration
  • escape-html - HTML security
  • formidable - File upload handling
  • geoip-lite - IP-based location services
  • he - HTML entity encoding/decoding
  • highlight.js - Syntax highlighting
  • markdown-it - Markdown parsing
  • mime-types - File type identification
  • otplib - 2FA/TOTP support
  • path-sanitizer - File path security
  • qrcode - QR code generation
  • sharp - Image processing
  • snowtransfer - Discord REST API library
  • ua-parser-js - Browser and OS detection
  • uuid - Unique identifier generation
  • ws - WebSocket communication
