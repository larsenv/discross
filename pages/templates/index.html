What is Discross?

A work-in-progress unofficial Discord relay (that acts like a client) aiming to support all platforms that support HTML. It uses webhooks and a bot so it doesn't need your user token, which would allow anyone to control your account. However, this means it only works on servers with the bot.

How to Use Discross

All of these steps must be performed on a modern device that can access Discord.com

  1. Use the link to add Discross to your server. Not necessary if you are already on a server with the bot.
  2. Type ^connect on the server to get your verification code
  3. Go to discross.net
  4. Register on the website with that code
  5. Click the "update your server list" button to authorize Discross with your Discord account (only required on first login or when you have added/removed servers)

Now you can use Discross on all of your devices by logging into the account you have just made!
