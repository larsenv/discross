What is Discross?

Discross is a universal Discord client designed to work on any device with a basic HTML web browser. Access Discord on retro consoles, old computers, and modern devices.

How to Use Discross

All of these steps must be performed on a modern device that can access Discord.com

  1. Use the link to add Discross to your server
  2. Type ^connect on the server to get your verification code
  3. Go to discross.net
  4. Register on the website with that code
  5. Click the "Update Server List" button to authorize Discross with your Discord account (only required on first login)

Now you can use Discross on all of your devices by logging into the account you have just made!
