What is Discross?

Discross is a work-in-progress unofficial Discord relay that acts like a client, specifically designed to support all platforms that support HTML. This means you can access Discord on devices that normally wouldn't be able to run the official Discord client.

It uses webhooks and a bot so it doesn't need your user token, which would allow anyone to control your account. Instead, you create a dedicated application that only has the permissions you give it.

Universal Compatibility

Works on any device with HTML support, including retro gaming consoles, legacy browsers, and embedded systems.

Lightweight Design

Optimized for low-bandwidth connections and minimal system requirements while maintaining core Discord functionality.

Open Source

Fully open source project hosted on GitHub. Contribute to development or host your own instance.

Retro Gaming Support

Tested and working on Nintendo Wii, PlayStation 3, Nintendo 3DS, and many other gaming platforms.

Getting Started

To get started with Discross, you'll need to add the bot to your Discord server and create an account. Want to try before connecting? Visit our Discord server at discord.gg/YrPG9krvVF and test it out without authentication.

You can revoke access at any time by visiting your Discord Developer Portal and deleting the Discross application.
